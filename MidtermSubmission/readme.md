# TD3BC AI for DareFightingICE

## Installation:
- Install python 3.8
```
pip install -r requirements.txt 
```
## Run the test
- Run the ```run-windows-amd64.bat``` file to start the DareFightingICE-6.0.2
```
python test.py
```
